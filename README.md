## 📂 Projects List

### 1. 💰 Personal Budget Tracker

A program designed to systematically record and track income and expenses.

* **Key Learning:** File Handling (CSV), Basic Data Types, User Input Validation.
* **Functionality:** Add income/expenses, calculate outstanding balance, and generate financial reports.

### 2. ⏰ Interactive Quiz Game

A time-limit-based quiz game where players answer questions under a countdown.

* **Key Learning:** `time` module, List of Dictionaries, Control Flow.
* **Functionality:** Displays countdown timer, measures elapsed time, and calculates the final score.

### 3. 📝 To-Do List Manager

A program to manage daily tasks and organize them based on priority levels.

* **Key Learning:** Custom Delimiters, Lambda Functions (Sorting), `datetime` module.
* **Functionality:** Add tasks, sort tasks by priority, and delete completed tasks.

### 4. 🌦️ Weather Data Analyzer

A data analysis tool that processes and analyzes temperature data by city.

* **Key Learning:** Data Grouping, Statistical Functions (`max`, `min`, `avg`), Error Handling.
* **Functionality:** Reads data from a CSV file and generates summary reports grouped by city.

### 5. 📱 Contact Book Manager (OOP)

A contact management application built using Object-Oriented Programming (OOP) concepts.

* **Key Learning:** Classes & Objects, Constructor (`__init__`), Data Synchronization.
* **Functionality:** Save, search, and update contact information.
